# Week 3

## 目標

本週計畫主要為整理素材以及設計 MyNTU 的主頁面。  
因此將工作分為四項：撰寫網頁內文、設計 HTML/CSS 骨幹、設計 CSS 動畫、製作手機版 CSS。  

## 使用上課技術

* HTML attributes
* title, h1, p tag
* a tag hyperlink
* input (text) box
* div, span, header, section, aside tag
* CSS font-family
* CSS color, background-color
* CSS class selector
* CSS margin, padding, border
* CSS display, position
* CSS box-sizing, width, height
* CSS z-index

## 使用額外技術

* CSS Variable
* CSS transition
* CSS Grid / Flex
* CSS Hover
* JS event listener
